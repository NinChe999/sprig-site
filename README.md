# Sprig

Landing page for **Sprig** — a monthly subscription box of better-ingredient dog treats and durable toys.

A single static `index.html` (HTML + CSS + a little vanilla JS, no build step).

## Run locally

Just open `index.html` in a browser, or serve it:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy with GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, choose your `main` branch and the `/ (root)` folder.
4. Save. Your site publishes at `https://YOUR_USERNAME.github.io/sprig-site`.

## Structure

- `index.html` — the entire site (nav with mega-dropdowns, hero, monthly box, how-it-works, why-sprig, reviews, pricing, FAQ, footer)

## Notes

- Icons in the nav are emoji placeholders; swap for SVGs when brand assets are ready.
- Pricing, copy, and links are placeholders.
